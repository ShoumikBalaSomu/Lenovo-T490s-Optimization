## 📖 Overview
A curated collection of optimization scripts and configuration files specifically tailored for the **Lenovo ThinkPad T490s**. This project aims to maximize performance, battery life, and user experience on both Windows 11 and Fedora Linux.

## ✨ Key Optimizations
- 🔋 **Power Management**: TLP configurations for Fedora to extend battery life.
- 🔊 **Audio Enhancement**: Scripts to enable 100% volume in GNOME and fix common audio issues.
- ⌨️ **Input & Keyboard**: Support for OpenBangla Keyboard and custom key mappings.
- 🚀 **Performance**: Windows 11 debloat scripts and registry tweaks.
- 🖥️ **Display**: Optimized scaling and color profiles.

## 🚀 How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shoumikbalasomu/Lenovo-T490s-Optimization.git
   cd Lenovo-T490s-Optimization
   ```

2. **Select your OS**:
   - Navigate to the `Linux/` or `Windows/` directory.
   - Follow the specific instructions in the sub-directory READMEs.

3. **Run the scripts**:
   - For Linux: `sudo ./optimize.sh`
   - For Windows: Run the `.ps1` or `.reg` files as Administrator.

## 🛠️ Requirements
- Lenovo ThinkPad T490s.
- Windows 11 or Fedora (Workstation/Silverblue).
